Ícones

Uma das práticas mais comuns hoje é utilizar SVG como fonte de ícones.

Uso direto do SVG

A primeira forma mais simples é colocar o conteúdo do SVG direto no lugar onde se deseja renderizar o ícone.

Uso referenciado do SVG

Uma segunda forma, para reaproveitar o mesmo SVG, é definindo símboloes e referenciando-os.

Via CSS

Uma terceira forma é utilizando o pseudo-elemento :hover e usando como o SVG como background
